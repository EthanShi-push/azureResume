I leveraged Java SpringBoot and Maven to build robust development environments, while using MyBatis-Plus to automate database operations. Additionally, I executed JUnit tests and used MockMvcBuilders to validate responses and ensure code reliability.

Alongside my backend work, I’ve self-learned Python, which led me to explore various projects. I built a 2-D multiplayer shooting game with Pygame, where I implemented collision detection algorithms and streamlined character movement animations. These projects reflect my love for experimenting with technology and constantly pushing my boundaries.

I’ve also started exploring cloud technologies in recent months. In one project, I built a portfolio website using Azure services such as Cosmos DB and Azure Functions, introducing me to serverless architectures and cloud database management. I’ve earned the Azure Fundamentals certification and am currently preparing for the Azure 204 certification to further expand my expertise in cloud development.

I’m excited to apply my skills in backend and cloud development to new challenges, and I’m eager to continue growing in the ever-evolving tech world.
